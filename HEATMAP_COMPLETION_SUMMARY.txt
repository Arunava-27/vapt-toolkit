## 💡 How to Use

### For End Users
1. Open a project in the VAPT toolkit
2. Click the **"🔥 Risk Heat Map"** tab
3. Select your desired view mode:
   - By Target: See vulnerability distribution across targets
   - By Time: Track progress over time
   - By Severity: View severity distribution
   - By Type: Analyze vulnerability types
4. Adjust filters as needed
5. Hover for details, export for sharing

### For Developers
1. Import the HeatMapGenerator:
   ```python
   from scanner.reporters.heatmap_generator import HeatMapGenerator
   gen = HeatMapGenerator()
   ```

2. Use the API endpoints:
   ```bash
   curl "http://localhost:8000/api/reports/heatmap/by-target?projectId=proj-123"
   ```

3. Integrate the React component:
   ```jsx
   import RiskHeatMap from "../components/RiskHeatMap";
   <RiskHeatMap projectId={id} />
   ```
